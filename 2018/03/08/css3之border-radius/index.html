border-radius => 边框半径
css3中专门针对圆角边框增加的属性

border-radius属性定义

  • border-top-left-radius:左上角
  • border-top-right-radius:右上角
  • border-bottom-right-radius:右下角
  • border-bottom-left-radius:左下角
  • inherit:表示四个值都从父元素计算值继承。

    border-radius(简写)

    border-radius是一种缩写方法，如果反斜线 “/”存在，“/”前面的值是设置元素圆角的水平半径，“/”后面的值是设置元素圆角的垂直方向的半径，如果没有“/”圆角的水平和垂直方向的半径值相等。

    border-radius:{1}设置一个值4个圆角相等


    border-radius:{2}设置两个值

    那么top-left和bottom-right取第一个值，top-right和bottom-left取第二个值，如图

    border-radius:{3}设置三个值

    第一个值设置对应top-left，第二个值对应top-right和bottom-left,第三个值对应bottom-right，如图

    border-radius:{4}设置四个值，顺时针计算

    第一个值为top-left ==> top-right==> bottom-right==> borttom-left
    注意border-radius的属性主要包含2个值
  • none 默认值，表示元素没有圆角。
  • 数值加单位，不能为负数
  • 设置没有圆角设置border-radius的值为0
