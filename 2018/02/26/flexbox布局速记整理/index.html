flexbox布局速记整理

发表于 2018-02-26 | 分类于 前端 | 阅读次数

flexbox 属性集合

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

父级（外层容器）

flex-direction

属性让你决定Flex项目如何排列，横向（水平）或者纵向（垂直）。

  • row 横向（默认值）
  • column 纵向
  • row-reverse 横向相反
  • column-reverse 纵向相反

flex-wrap

控制排列是否换行排列

  • wrap 换行
  • nowrap 不换行
  • wrap-reverse 换行（相反排列，最后一行在最上方）

缩写

flex-flow是flex-direction和flex-wrap两个属性的速记属性

  • flex-flow: row nowrap 横向不换行
  • flex-flow: column wrap 纵向换行
  • flex-flow: column nowrap 纵向不换行

justify-content

在主轴方向上对齐方式（横向）

  • flex-start 左对齐 （默认）
  • flex-end 右对齐
  • center 居中
  • space-between 两端对齐
  • space-around 项目两侧间距相等

align-items

在交叉轴方向上对齐方式（纵向）

  • flex-start 交叉轴起点，顶部
  • flex-end 交叉轴终点，底部
  • center 交叉轴中间
  • baseline 项目第一行文字基线
  • stretch 充满容器（默认）

align-content

属性用于多行的Flex容器（纵向）

  • flex-start 交叉轴起点，顶部
  • flex-end 交叉轴终点，底部
  • center 交叉轴中间
  • space-between 交叉轴上线对齐，中间间隔平均
  • stretch 充满容器（默认）

子级（子容器）

align-self

控制子容器

  • auto 继承align-items，若无则为 stretch
  • flex-start 交叉轴起点，顶部
  • flex-end 交叉轴终点，底部
  • center 交叉轴中间
  • baseline 项目第一行文字基线
  • stretch 拉伸充满容器（默认）

order

排序越小越靠前， 可以为负数

flex

控制项目容器的显示

  • flex-grow: 0 || 1; 容器不会放大
  • flex-shrink: 1 || 0; 容器可以放大缩小，为0 不会缩小
  • flex-basis: auto || (数值); auto（默认项目大小） || 指定的数值

flex速记

flex是flex-grow、flex-shrink和flex-basis三个属性的简写。
flex: 0 1 auto => flex-grow: 0; flex-shrink: 1; flex-basis: auto;
